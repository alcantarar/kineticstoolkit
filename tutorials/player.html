Player¶

The Player class allows visualizing markers, rigid bodies and segments in movement and in 3D using a keyboard and mouse-driven user interface.

This tutorial is still incomplete. Please see the kinematics tutorial, which make other uses of the Player class.

In [1]:
import ktk

Loading sample data¶

In this tutorial, we will load propulsion data from a sprint in Wheelchair Basketball, as recorded using an optoelectronic system (Optitrack). Data were preprocessed in Matlab.

In [2]:
kinematics = ktk.loadmat('data/inversedynamics/basketball_kinematics.mat')
kinematics = kinematics['kinematics']

The player can be instanciated to show markers:

In [3]:
pl = ktk.Player(markers=kinematics['Markers'], target='centroid')

It is possible to colorize markers, using one of these letters:

- (r)ed
- (g)reen
- (b)lue
- (y)ellow
- (m)agenta
- (c)yan
- (w)hite
