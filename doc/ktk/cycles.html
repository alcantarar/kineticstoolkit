Module ktk.cycles

Identify cycles and time-normalize data.

This module in early development provides functions to identify cycles and time-normalize data.

def time_normalize(ts, event_name1, event_name2, n_points=100)

Time-normalize cycles in a TimeSeries.

This method time-normalizes the TimeSeries at each cycle defined by event_name1 and event_name2 on n_points. The time-normalized cycles are put end to end. For example, for a TimeSeries that contains three cycles, a time normalization with 100 points will give a TimeSeries of length 300. The TimeSeries' events are also time-normalized.

Parameters

ts : TimeSeries
The TimeSeries to analyze.
event_name1, event_name2 : str
The events that correspond to the begin and end of a cycle.
n_points : int (optional)
The number of points of the output TimeSeries (default is 100).

Returns

ts_copy : TimeSeries
A new TimeSeries where each cycles has been time-normalized.
